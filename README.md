# 🏠 개발하게 된 계기

<aside>
아버지께서 에어컨 설치 기사십니다. 개인 사업자이면서도 협력 업체로써 일을 하시는데, 협력업체에서는 앱이 있어서 관리가 쉽지만, 개인 사업자로써 내역을 확인하기 위해서는 메시지를 모두 확인해야하는 어려움이 있으셨습니다.
그래서 이 문제를 해결하고자 앱을 개발하게 되었습니다.
</aside>
